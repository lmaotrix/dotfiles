# My Dotfiles

Configuration files for my Arch linux WSL terminal environment, managed with GNU Stow.

## Setup instructions
1. Clone the repo:
```bash
git clone https://github.com/lmaotrix/dotfiles.git ~/dotfiles
```
2. Run the install script:
```bash
cd ~/dotfiles 
./install.sh
```
3. Source your shell config:
```bash
source ~/.zshrc 
```

## Dependencies
- Arch linux (WSL)
- zsh 
- neovim
- starship prompt
- tmux 
- git 
- GNU stow
